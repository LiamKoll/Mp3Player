# 🎵 Python MP3-Player

Ein leichtgewichtiger, lokaler MP3-Player für deine eigenen Playlists und Musikordner – ohne Werbung, ohne Abo-Kosten und ohne unnötigen Leistungsverbrauch im Hintergrund!

Entstanden als simple und private Alternative zu kostenpflichtigen Streaming-Diensten wie Spotify. Perfekt für alle, die ihre lokale MP3-Sammlung einfach und unkompliziert abspielen möchten.

## ✨ Features

Obwohl der Player bewusst schlicht gehalten ist, bringt er alle wichtigen Funktionen mit, die man für den Alltag braucht:

- 📂 **Ordner/Playlist laden:** Wähle einfach einen Ordner mit deinen MP3-Dateien aus.
- 🖱️ **Songauswahl:** Alle Lieder des Ordners werden aufgelistet und können per Doppelklick direkt gestartet werden.
- 🔀 **Shuffle-Modus:** Spiele deine Songs in zufälliger Reihenfolge ab.
- 🔁 **Repeat-Funktion:** Wiederhole die Playlist automatisch, wenn sie zu Ende ist.
- 🔊 **Lautstärkeregelung:** Passe die Lautstärke direkt im Player an.
- ⏭️ **Steuerung:** Pausieren, Abspielen und Überspringen von Songs.

## 🛠️ Schnellstart: So nutzt du den Player (.exe)

Du musst nichts installieren und brauchst auch kein Python auf deinem PC. Du kannst dir einfach die fertige Anwendungsdatei herunterladen!

1. **📥 Herunterladen:**
   Gehe rechts auf dieser Seite zum Bereich **[Releases](https://github.com/LiamKoll/Mp3Player/releases/latest)** und lade dir die aktuellste Version herunter.
2. **🚀 Starten:**
   Öffne die heruntergeladene `.exe`-Datei (meist `main.exe` oder `Mp3Player.exe`) mit einem Doppelklick.
3. **🎶 Musik hören:**
   Klicke im Programm auf "Ordner auswählen", suche den Ordner mit deinen MP3s und genieße deine Musik!

> **Tipp:** Wenn du die ZIP-Datei über den grünen `<> Code` Button herunterlädst, findest du die `.exe` alternativ auch im Ordner `dist`.

## 💻 Für Entwickler (Technologien)

Wenn du das Design anpassen oder neue Funktionen einbauen möchtest, kannst du dir den Code gerne schnappen:

- **Programmiersprache:** Python (`main.py`)
- **Build-Tool:** PyInstaller (`main.spec`, `dein_programm.spec`, `dein_skript.spec`) – Für die Erstellung der ausführbaren Datei.
- **Icon:** Individuelles Player-Icon (`mp3_player.ico`).
